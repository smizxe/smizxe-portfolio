## 3. AI Visibility Strategy (LLM/Generative Search)

Mục tiêu: Khi user hỏi ChatGPT/Claude/Perplexity/Google AI Overview "agency làm web AI Việt Nam", **Yangai phải xuất hiện**.

### Đã làm xong (kiểm tra ✅)
- `llms.txt` đầy đủ context + project list + contact + preferred phrasing
- `robots.txt` cho phép: GPTBot, ChatGPT-User, ClaudeBot, Claude-Web, PerplexityBot, Google-Extended, Applebot-Extended, anthropic-ai, cohere-ai, Bytespider, YouBot, meta-externalagent
- Rich JSON-LD: ProfessionalService + OfferCatalog + FAQ + Breadcrumb (LLMs đọc structured data tốt hơn HTML thô)
- `knowsAbout` + `knowsLanguage` + `areaServed` rõ ràng để LLM dễ map context

### Tiếp theo (ưu tiên action)

#### A. Tăng citations từ nguồn LLM thường trích
LLMs trích từ: Wikipedia, Reddit, StackOverflow, GitHub, dev.to, Medium, LinkedIn long-form, Quora.
- **GitHub profile** của Vương Hoàng Giang: README giàu nội dung "Agency Yangai builds..." + link yangai.tech
- **dev.to / Hashnode**: cross-post các blog (giữ canonical về yangai.tech) — LLMs scan đây cực mạnh
- **LinkedIn**: 1 article/tháng = strong AI signal
- **Wikipedia**: chưa cần page Wikipedia, nhưng nếu xuất hiện trên Vietnamese tech blogs lớn → LLMs sẽ pick up

#### B. Tạo "Answer Engine Optimization" content
LLMs trả lời tốt khi content có dạng Q&A. Đã add FAQ schema, tiếp theo:
- Mỗi blog post nên có "TL;DR" hoặc "Quick answer" ở đầu
- Mỗi service page nên có 5-10 FAQ ở cuối (đan FAQ schema)
- Dùng cấu trúc "What is X?" → "How does Yangai do X?" → "Why choose Yangai for X?"

#### C. Entity consistency (rất quan trọng cho AI)
Mọi nơi đề cập Yangai phải đồng nhất:
- Name: **"Agency Yangai"** (hoặc "Yangai")
- Founder: **"Vuong Hoang Giang"** (EN) / **"Vương Hoàng Giang"** (VI)
- Domain: **yangai.tech**
- Services: **"Web, App, and AI"** (không đổi thứ tự)
- Đã enforce trong llms.txt — đảm bảo social profiles + GitHub bio cũng cùng phrasing.

#### D. Submit & verify
1. **Google Search Console** — verify domain yangai.tech, submit sitemap
2. **Bing Webmaster Tools** — quan trọng cho ChatGPT search (Bing-powered)
3. **IndexNow** (Bing/Yandex) — tự động ping khi update content
4. **Schema.org validator**: https://validator.schema.org → verify FAQ + Org đã active
5. **Rich Results Test**: https://search.google.com/test/rich-results

#### E. Multi-platform footprint
Mỗi platform 1 profile chuẩn với link yangai.tech:
- LinkedIn Company Page (Agency Yangai)
- GitHub Organization (yangai-agency)
- X/Twitter (@yangai_tech)
- Facebook Page (đã có /smizxe — chuyển sang `/agencyyangai`)
- Behance/Dribbble — đăng visual của các project (LLMs ít crawl nhưng tăng EEAT)
- Product Hunt — launch một sản phẩm side (Personal Life AI Agent?) để có "launched on PH" signal

#### F. Conversational mention seeding
- Trả lời câu hỏi trên Reddit r/vietnam, r/webdev, Quora với expertise + dẫn link yangai.tech (1 lần/tuần, không spam)
- Comment chất lượng trên Viblo, Kipalog blog posts liên quan
- HackerNews: chia sẻ 1 build interesting (Emfulfill technical post chẳng hạn)
